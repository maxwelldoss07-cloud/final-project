This is a simple program I put together to analyze music taste. The user selects a few songs or searches for songs and selects them from the larger library, and then they press the "Analyze" button to do some math and receive their selected songs' genres in highest to lowest percentage. The program then selects other songs from the highest percentage genre and shows the user a button to save them to the browser. The user can also remove saved songs from selection.
Here is the site it has hosted: https://maxwelldoss07-cloud.github.io/Find-My-Genre/
